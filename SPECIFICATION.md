Project Objectives

Objective 1 – Develop an AI Travel Assistant
Create an application that generates a personalised travel itinerary using an AI language model.
Generate recommendations based on the user's travel preferences.

Objective 2 – Collect User Travel Information
Allow the user to enter a travel destination.
Allow the user to select the start and end dates of their trip.
Allow the user to enter an approximate budget.
Allow the user to select one or more interests (e.g. sightseeing, food, beaches or hiking).
Allow the user to choose a travel style (e.g. budget, luxury or adventure).

Objective 3 – Generate a Personalised Itinerary
Produce a day-by-day itinerary based on the user's input.
Include suggested activities and places to visit for each day.
Tailor the recommendations to the user's selected interests, budget and travel style.

Objective 4 – Display Live Travel Information
Retrieve weather information for the selected destination using an external API.
Display the weather forecast alongside the generated itinerary.
Notify the user if weather information cannot be retrieved.

Objective 5 – Create a Graphical User Interface
Provide input fields for all required travel information.
Include a button to generate the itinerary.
Display the completed itinerary within the application.
Allow the user to generate multiple itineraries without restarting the program.

Objective 6 – Validate User Input
Prevent an itinerary from being generated if required information has not been entered.
Display an appropriate error message for invalid or missing input.
Handle API errors without causing the application to crash.

Objective 7 – Produce Well-Structured Software
Separate the program into multiple functions, each performing a specific task.
Store API keys securely using a .env file.
Organise the source code into appropriate files where necessary.

Objective 8 – Test the Application
Test each major feature individually.
Test the application using different destinations, budgets and interests.
Correct any faults identified during testing.

Objective 9 – Produce a Complete Working Application
Ensure all planned features are implemented.
Ensure the application runs without runtime errors during normal use.
Produce documentation explaining how to install and run the application.
